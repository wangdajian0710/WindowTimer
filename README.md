# WindowTimer

Windows 窗口计时监视器 - 基于 PowerShell + WinForms 的任务栏托盘计时工具。

> ⚠️ 这是 **TaskbarTimer** 的升级版，两个项目独立维护。

## 功能

- ⏱️ 秒表计时（开始/暂停/重置）
- 📋 分段记录
- 📌 任务栏固定位置
- 🪟 监测指定窗口的活跃时间
- 🎨 现代化 UI 界面

## 使用方法

右键点击 `WindowTimer.ps1` → 选择 **"使用 PowerShell 运行"**。

## 系统要求

- Windows 7 / 8 / 10 / 11
- PowerShell 5.1（系统自带）

## 退出方式

右键任务栏托盘图标，选择"退出"。
